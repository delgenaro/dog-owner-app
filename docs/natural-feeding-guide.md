# Guia de Alimentacao Natural (AN) para Caes

Fontes: Mascote Fit, Petlove, Cobasi, Cachorro Verde, Perfectly Rawsome, AAFCO, FEDIAF.

## O que e Alimentacao Natural?

Dieta baseada em ingredientes frescos e minimamente processados, formulada para atender as necessidades nutricionais dos caes. Diferente de "restos de comida" - e uma dieta planejada com orientacao veterinaria.

### Tipos de AN

| Tipo | Descricao | Indicado para |
|---|---|---|
| AN Cozida | Alimentos levemente cozidos | Iniciantes, caes com saude sensivel |
| BARF (crua c/ ossos) | Carnes, ossos e orgaos crus | Tutores experientes |
| AN Crua sem ossos | Igual BARF, mas com calcio suplementar | Quem prefere evitar ossos |
| Prey Model | 100% animal (sem vegetais/frutas) | Dieta mais proximal a natureza |

## Proporcoes Recomendadas

### AN Cozida (padrao brasileiro)

| Grupo | % da dieta | Exemplos |
|---|---|---|
| Carnes desossadas | 30% | Frango, bovino, peixe, ovo |
| Vísceras | 5% | Figado, coracao, rim, moela |
| Carboidratos | 35% | Batata-doce, mandioquinha, arroz |
| Vegetais | 30% | Cenoura, abobrinha, brocolis, chuchu |
| Frutas | 10-15% extra | Maca, banana, melancia (petisco) |

### BARF (Biologically Appropriate Raw Food)

| Grupo | % da dieta |
|---|---|
| Carne muscular | 70% |
| Osso cru comestivel | 10% |
| Figado | 5% |
| Outros orgaos secretores | 5% |
| Vegetais | 7% |
| Sementes/nozes | 2% |
| Frutas | 1% |

### Prey Model
- 80% carne muscular
- 10% osso cru
- 5% figado
- 5% outros orgaos

## Quantidade Diaria

Regra geral: 2% a 4% do peso corporal por dia.

| Perfil | % do peso |
|---|---|
| Adulto sedentario / obeso | 2% |
| Adulto moderado | 2.5 - 3% |
| Adulto ativo | 3 - 4% |
| Filhote | 3 - 6% (fracionado) |
| Idoso | 2 - 2.5% |

## Alimentos Permitidos

### Proteinas
- Frango, bovina, suina (sempre bem cozida)
- Peixes (tilapia, merluza, sardinha - sem espinhas)
- Ovos (cozidos ou crus com procedencia)
- Cordeiro, coelho, peru

### Vísceras (5% da dieta)
- Figado (max 5% - risco hipervitaminose A)
- Coracao (rico em taurina)
- Rim, moela, baco, pulmao

### Carboidratos
- Batata-doce, mandioquinha, inhame
- Arroz integral, aveia, quinoa
- Mandioca cozida

### Vegetais
- Cenoura, abobrinha, brocolis, couve-flor
- Chuchu, vagem, beterraba, espinafre
- Abobora, pepino

### Frutas (petiscos, moderacao)
- Maca (sem sementes), banana, pera
- Melancia, melao, mamao
- Mirtilo, morango, framboesa
- Manga, kiwi, goiaba

### Gorduras Boas
- Oleo de coco extra virgem
- Azeite de oliva extra virgem
- Oleo de peixe (omega-3)
- Oleo de linahca

## Alimentos Proibidos

| Alimento | Risco |
|---|---|
| Chocolate | Teobromina - toxica |
| Cebola e alho | Danificam hemacias |
| Uva e uva passa | Insuficiencia renal |
| Carambola | Toxica para rins |
| Macadamia | Neurotoxica |
| Ossos cozidos | Lascam - perfuracao intestinal |
| Cafe, alcool | Estimulantes toxicos |
| Xilitol (adocante) | Queda de acucar fatal |
| Abacate (casca, caroco) | Persina - toxica |
| Sal em excesso | Desidratacao |
| Massa crua (fermento) | Fermenta no estomago |

## Suplementacao Obrigatoria

### Calcio (CRITICO!)
Carne tem muito fosforo e pouco calcio. Sem osso, suplementar e obrigatorio.
- Casca de ovo em po: ~1 colher cha / kg de alimento
- Farinha de ossos ou suplemento mineral (ex: Food Dog)

### Omega-3
- Oleo de peixe (salmao, sardinha)
- ~1000mg / 10kg / dia

### Outros
- Vitamina E: antioxidante
- Complexo B: especialmente se dieta cozida
- Taurina: importante para coracao (dieta cozida)

## Transicao da Racao para AN

1. Dia 1-3: 75% racao + 25% AN
2. Dia 4-6: 50% racao + 50% AN
3. Dia 7-9: 25% racao + 75% AN
4. Dia 10+: 100% AN

## Higiene e Seguranca
- Carnes congeladas por min. 3 dias a -18C (se servidas crus)
- Utensilios separados para carne crua
- Nao deixar AN em temperatura ambiente > 2h
- Armazenar em porcoes diarias no congelador
